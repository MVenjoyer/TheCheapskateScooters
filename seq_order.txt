@startuml
title Процесс аренды самоката

actor Пользователь
participant "Веб-интерфейс" as UI
participant "API Сервер" as API
database "База данных" as DB

== 1. Поиск точки ==
Пользователь -> UI : Открывает карту
UI -> UI : Загружает карту с точками аренды
UI -> UI : Показывает точки с доступными самокатами
Пользователь -> UI : Выбирает точку на карте
UI -> UI : Открывает балун с информацией
Пользователь -> UI : Нажимает "Подробнее о точке"

== 2. Выбор самоката ==
UI -> UI : Переход на страницу точки
UI -> UI : Отображение списка самокатов в точке
Пользователь -> UI : Выбирает доступный самокат
Пользователь -> UI : Нажимает "Арендовать"

== 3. Аренда ==
UI -> API : POST /api/rent/<scooter_id>/
API -> DB : Проверка доступности самоката
DB -> API : Самокат доступен
API -> DB : Создание записи аренды
API -> DB : Обновление статуса самоката
API -> UI : Ответ: успешно
UI -> Пользователь : "Самокат арендован"

@enduml
